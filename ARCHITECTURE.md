# Architectural Constitution

This project follows these immutable rules:

1. Use Tailwind CSS
2. Mobile First Design
3. Use TypeScript
4. No inline styles
5. Use meaningful variable names
6. Comment complex logic
7. Keep functions under 20 lines
8. Use async/await over promises
9. Animation Library: Incorporate Framer Motion for all transitions and animations.
10. Spacing: Use a consistent spacing scale (multiples of 4px) for all margins and padding.
